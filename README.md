# Holberton School Dashboard

This project is a basic React application created with Vite as part of the Holberton School React introduction curriculum.

## Project Structure

- `task_0/dashboard`: Initial dashboard setup.
  - `src/App.jsx`: Main application component.
  - `src/App.css`: Application styles.
  - `setupTests.js`: Testing environment configuration.
  - `.babelrc`: Babel configuration.
  - `fileTransformer.js`: Jest asset transformer.

## Setup and Installation

1. Navigate to the dashboard directory:
   ```bash
   cd task_0/dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Assets:
   Place the following files in their respective directories:
   - `holberton-logo.jpg` into `src/assets/`
   - `close-button.png` into `src/assets/`
   - `favicon.ico` into `public/`

## Available Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs ESLint check.
- `npm run test`: Runs the test suite using Jest.
